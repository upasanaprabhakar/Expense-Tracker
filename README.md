<section>
  <h2 style="font-size: 44px; margin-top: 40px;">✨ Features</h2>
  <ul style="font-size: 24px;">
    <li>📊 Real-time balance, income, and expense calculation</li>
    <li>🗂️ Category-wise transaction filtering</li>
    <li>📈 Interactive transactions chart</li>
    <li>➕ Add and delete transactions dynamically</li>
    <li>💾 Persistent data storage with <b>localStorage</b></li>
    <li>📱 Fully responsive design for all devices</li>
